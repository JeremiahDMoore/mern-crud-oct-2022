# To Do App made with MERN
## MongoDB | Express | React | Node


Simple app to handle your todos. Creates Todo item, Reads from the Database, Updates Todo item, Deletes Todo item.

## Dependencies
```
"cors": "^2.8.5",
"dotenv": "^10.0.0",
"express": "^4.17.1",
"mongoose": "^6.0.8",
"nodemon": "^2.0.13"
"axios": "^0.24.0",
"node-sass": "^6.0.1",
"react": "^17.0.2",
"react-dom": "^17.0.2",
"react-router-dom": "^5.3.0",
"react-scripts": "4.0.3",
"web-vitals": "^1.1.2"
```

## Installation
```
cd server
npm install
cd ../client
npm install

npm start
```
server runs on
```
http://localhost:4000
```
client runs on
```
http://localhost:3000
```
